# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/63b50684-3af6-4936-b4ce-dd9b6f8babfd

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `MIMO_API_KEY` in [.env.local](.env.local) to your Xiaomi MiMo API key
3. Run the app:
   `npm run dev`
